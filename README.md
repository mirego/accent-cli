# accent

Sync, merge and export with the Accent V2 API. Just a fancy cUrl wrapper.

## Install

```bash
$ brew update
$ brew tap mirego/formulae
$ brew install accent
```

## Usage Example

```bash
$ accent sync \
    --api-url=https://api.accent.reviews \
    --api-key=test123 \
    --input-file=locale.json \
    --language=fr
```

## Development

```bash
$ make build
$ ./bin/accent --help
```
